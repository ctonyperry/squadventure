## For Now

To get started immediately:

```bash
# Install and build
pnpm install
pnpm build

# Start the game
pnpm dev
```

Type naturally to interact with the AI Dungeon Master. Try things like:
- "I look around the tavern"
- "I approach the bartender and ask about rumors"
- "I draw my sword and attack the goblin"

Use `/help` to see available commands.
