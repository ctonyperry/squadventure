# SRD Content

This document lists the D&D 5th Edition System Reference Document (SRD) content included in Squadventure.

## What is the SRD?

The [System Reference Document](https://dnd.wizards.com/resources/systems-reference-document) is a subset of D&D 5e rules published by Wizards of the Coast under the Open Gaming License (OGL). It provides the core rules for D&D 5e that can be used freely.

## Included Classes

### Fighter
- **Hit Die:** d10
- **Primary:** Strength or Dexterity
- **Saves:** Strength, Constitution
- **Subclass:** Champion (level 3)

### Rogue
- **Hit Die:** d8
- **Primary:** Dexterity
- **Saves:** Dexterity, Intelligence
- **Subclass:** Thief (level 3)

### Wizard
- **Hit Die:** d6
- **Primary:** Intelligence
- **Saves:** Intelligence, Wisdom
- **Subclass:** School of Evocation (level 2)
- **Spellcasting:** Full caster

### Cleric
- **Hit Die:** d8
- **Primary:** Wisdom
- **Saves:** Wisdom, Charisma
- **Subclass:** Life Domain (level 1)
- **Spellcasting:** Full caster

## Conditions

All standard D&D 5e conditions are implemented:

| Condition | Key Effects |
|-----------|-------------|
| Blinded | Can't see, attack rolls have disadvantage |
| Charmed | Can't attack charmer |
| Deafened | Can't hear |
| Exhaustion | Cumulative penalties |
| Frightened | Disadvantage while source is visible |
| Grappled | Speed becomes 0 |
| Incapacitated | Can't take actions |
| Invisible | Attacks against have disadvantage |
| Paralyzed | Auto-fail STR/DEX saves |
| Petrified | Turned to stone |
| Poisoned | Disadvantage on attacks/checks |
| Prone | Disadvantage on attacks |
| Restrained | Speed 0, disadvantage on attacks |
| Stunned | Incapacitated, auto-fail STR/DEX |
| Unconscious | Incapacitated and unaware |

## Not Included

The following are NOT included (not part of SRD):
- Non-SRD subclasses (e.g., Battlemaster, Assassin)
- Non-SRD races (e.g., Aasimar, Firbolg)
- Non-SRD spells
- Setting-specific content
- Most magic items
