## Brief Description

The Player Guide will provide a welcoming introduction to tabletop RPGs for new players, including kids and parents playing together. It will cover:

### The Basics
What is a tabletop RPG? How does Squadventure work? What's a Dungeon Master?

### Your Character
Understanding race, class, and abilities in simple terms. What do all those numbers mean?

### Playing the Game
How to describe what you want to do. Combat basics. Using spells and abilities.

### Tips & Tricks
Common mistakes to avoid. How to have more fun. Creative problem-solving.

## For Now

If you want to start playing:

```bash
pnpm dev
```

Type naturally to interact with the AI Dungeon Master:
- "I look around" - to observe your surroundings
- "I talk to the bartender" - to start a conversation
- "I attack the goblin" - to begin combat

The DM will guide you through the rest!
